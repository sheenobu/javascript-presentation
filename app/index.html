Client Side Templates

What are client side templates?

Templates which run within the browser. They typically output HTML which can be inserted into the DOM.

What can they do for you?

  • Help decouple backend and frontend services.
  • Help simplify AJAX.
  • Reduce spaghetti code within browser scripting.

Architecture

TYPES OF JAVASCRIPT ENGINES

Procedural:

  • Compile Template.
  • Build our Context.
  • Render the template.
  • Use template results.

MVC/MVVM:

  • Mark up your HTML.
  • Write a controller/viewmodel in javascript.
  • Initialize the engine.
